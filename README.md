# Jsdelivr-CDN
利用jsdelivr进行静态文件的cdn加速服务
请在后台CDN加速域名里填写    
https://cdn.jsdelivr.net/gh/lonelypers/Jsdelivr-CDN@master/Akina/3.4.3    
3.4.3为主题版本号，根据实际填写。    
## Akina文件夹
进行Akina-for-Typecho静态文件加速    
## file 
一些文件    
## img
静态图片    
## CDN刷新
若链接想手动刷新时,把链接中的

https://cdn.jsdelivr.net/
替换成

https://purge.jsdelivr.net/
即可实时刷新
